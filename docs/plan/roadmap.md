## Phase 1: Quick Win (1-2주)

### 1.1 PRD 비교 기능
- **설명**: 버전 간 diff 뷰어 (이미 버전 히스토리 존재)
- **구현**: react-diff-viewer 또는 자체 구현
- **파일**: `src/features/prd/ui/version-diff.tsx`
- **난이도**: 중

### 1.2 PRD 공유 링크
- **설명**: 읽기 전용 공개 URL 생성
- **구현**:
  - DB: `prds.share_token` (UUID), `prds.is_public` (boolean)
  - Route: `/share/[token]` - 공개 PRD 뷰어
- **난이도**: 중

### 1.3 대시보드 분석
- **설명**: 생성된 PRD 통계 (템플릿별, 버전별, 기간별)
- **구현**:
  - API: `/api/analytics/prds`
  - UI: `src/features/analytics/ui/prd-stats.tsx`
- **난이도**: 쉬움

## Phase 4: AI 품질 고도화 (지속적)

### 4.1 PRD 섹션별 재생성
- **설명**: 특정 섹션만 AI로 다시 작성
- **구현**:
  - UI: 섹션별 "재생성" 버튼
  - API: 섹션 컨텍스트 유지하며 재생성
- **난이도**: 중

### 4.2 Few-shot 예제 추가
- **설명**: 고품질 PRD 예제를 프롬프트에 포함
- **구현**: 템플릿별 예제 PRD 3-5개 작성
- **난이도**: 중 (작성 노력 필요)

### 4.3 피드백 기반 학습
- **설명**: 수집된 평가 데이터로 프롬프트 개선
- **구현**:
  - 분석: 낮은 평가 PRD 패턴 분석
  - 개선: 프롬프트 A/B 테스트
- **난이도**: 어려움

### 4.4 다국어 확장
- **설명**: 일본어, 영어 외 추가 언어 지원
- **구현**: 프롬프트 + UI 다국어화
- **난이도**: 중
